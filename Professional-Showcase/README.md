# Professional GitHub Profile Visuals

> **Very beautiful, effective, and professional looking.**

// Created by AshrafMorningstar - https://github.com/AshrafMorningstar

## Overview

This project generates stunning, high-quality GIFs and videos for your GitHub profile. Built with [Remotion](https://www.remotion.dev/), it turns your GitHub statistics into a cinematic experience.

## Features

- **Premium Aesthetics**: Glassmorphism, Neon gradients, and smooth animations.
- **Dynamic Data**: Fetches real-time stats from GitHub API.
- **Customizable**: Easy to tweak colors and configurations.
- **Professional Branding**: Automatically includes your name and profile link.

## Compositions

- **Stats**: A general overview of your profile.
- **Languages**: Top languages used.
- **Start**: Contribution graph visualizations.
  -And many more...

## How to specificy your username

Edit `input.json`:

```json
{
	"usernames": ["AshrafMorningstar"]
}
```

## Running the Project

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the preview studio:

   ```bash
   npm start
   ```

3. Render the output:
   ```bash
   npm run render
   ```
